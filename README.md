## Инструкция

### Команды для компиляции верстки

#### Режим разработки

>Для создания локального сервера и сборкой проекта
>```
>npm run serve 
>```

>Для простой сборкой проекта
>```
>npm run build
>```

>Для очистки папки .dist
>```
>npm run clear
>```


#### Режим production
>Для создания локального сервера и сборкой проекта готового к продакшену
>```
>npm run serveProd
>```

>Для простой сборкой проекта
>```
>npm run prod
>```
